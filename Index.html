CSS Basics

CSS in full is Cascading Style Sheets and it is used to style web pages. It handles the look and feel part of a web page. CSS is combined with the markup languages HTML or XHTML

So What is CSS?

It is a language used for describing the presentation of a document written in amarkup language like HTML. It is a cornerstone technology of the world wide web. You can control the color of the text, the style of fonts, the spacing between paragraphs, what background images or colors are used and much more. CSS is easy to learn and understand but provides a powerful control over presentation of an HTML document
